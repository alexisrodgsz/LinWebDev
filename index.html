HTML5 expandend to 7 main models

  1. flow: the mayority
  2. Metadata - in the head of the doc script, style, title, noscript, meta, link, base
  3. Embedded: import other resources: iframe, img, math, object, svg, video, audio, canvas, embed
  4. Interactive: somekind user interaction: a, audio, button, embed, iframe, img*, input*, label, object*, select, textarea, video*,
  5. Heading: theheading of the section: all h1-6
  6. Phrasing: the text of the document a lot of inlevel features span, strong, italic, video, object, output
  7. Sectioning: defines the scoope: article, aside, section
